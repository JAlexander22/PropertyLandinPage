# Project Expansion & Custom Features

The original foundation and UI design of this project was inspired by and followed from a YouTube tutorial created by:

How to Web Dev

This repository was then expanded further with additional functionality, interaction systems, UI improvements, and Google Analytics tracking features.

The goal of these additions was to turn the original landing page into a more interactive portfolio project focused on:
- React frontend development
- user interaction tracking
- marketing analytics
- conversion-focused UI behaviour

---

# Features Added

## Interactive Property Modal
Added a clickable property modal allowing users to:
- open property details
- browse property images
- view additional property information

---

## CTA & Conversion Features
Added:
- “Contact Us” CTA button
- “Enquire Now” property buttons
- smooth scroll navigation to enquiry form

---

## Enquiry Form
Implemented a responsive enquiry form with:
- controlled React inputs
- dynamic property interest selection
- form validation
- lead interaction simulation

---

## Google Analytics 4 Integration
Integrated Google Analytics 4 event tracking using React GA4.

Tracked events include:
- `enquire_now_clicked`
- `form_submitted`

This allows interaction data such as:
- property engagement
- CTA clicks
- enquiry conversions
- user behaviour patterns

to be analysed inside GA4.

---

# Purpose Of The Additions

These additions were implemented primarily to support:
- frontend portfolio development
- marketing analytics portfolio work
- GA4 behavioural analysis practice
- user interaction tracking demonstrations

The project is intended as a frontend analytics-focused portfolio piece rather than a full production real estate platform.
